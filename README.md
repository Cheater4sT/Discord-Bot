# Discord Music Bot

A Discord bot that allows users to play music in voice channels using commands.  
The bot supports playing audio from links and provides basic music control features.

---

## Features

- Play music in a voice channel
- Support for audio links (YouTube, SoundCloud)
- Pause / resume playback
- Skip tracks
- Stop playback and leave the channel
- Queue support

---

## Technologies Used

- Python
- discord.py
- yt-dlp
- FFmpeg
